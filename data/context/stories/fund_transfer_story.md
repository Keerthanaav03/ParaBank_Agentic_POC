# Jira Story

Story ID:
BANK-101

Title:
Fund Transfer Between Accounts

Story:
As a banking customer,
I want to transfer funds between my accounts,
So that I can manage my money efficiently.

Acceptance Criteria:

1. User should login successfully
2. User should navigate to Transfer Funds
3. User should select source account
4. User should select destination account
5. User should enter transfer amount
6. User should submit transfer
7. Success confirmation should appear

Negative Scenarios:

1. Transfer amount cannot be empty
2. Transfer amount cannot be zero
3. Transfer amount cannot exceed available balance
4. Source and destination account cannot be same

# Banking Domain Knowledge

## Common Banking Terms

- Savings Account
- Current Account
- Beneficiary
- Payee
- Transaction ID
- Available Balance
- Account Number
- Loan Eligibility
- Fund Transfer
- Bill Payment

---

## Banking Validation Standards

- Account numbers should be unique
- Monetary values cannot be negative
- Transactions require confirmation
- Sensitive operations require authentication

- Use explicit waits
- Avoid Thread.sleep()
